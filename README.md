A full-stack web application inspired by Airbnb, built using the MERN stack.
Users can explore listings, add new places, and manage their own listings.


🚀 Live Demo

https://delta-project-a949.onrender.com/listings


🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap, EJS
Backend: Node.js, Express.js
Database: MongoDB Atlas
Deployment: Render
Others: Cloudinary (for image upload), Map API
MVC Framework



✨ Features
🔐 User Authentication (Login / Register)
🏡 Create, Edit & Delete Listings
📸 Image Upload using Cloudinary
⭐ Add Reviews & Ratings
📍 Location-based listings
🌐 Fully deployed on cloud
